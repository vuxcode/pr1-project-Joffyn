Catch-Colored-Candy-Cubes

This is a simple game where you can navigate a 2D enviroment with obstacles to try and catch different colored squares that try to run away from you. The game features a way to calculate field of view called "Symmetric Shadowcasting". The implementation is based on Albert Fords version from --his website.--
